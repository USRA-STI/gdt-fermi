Fermi GBM Localizations using the DoL (:mod:`gdt.missions.fermi.gbm.localization.dol`)
**************************************************************************************

The ``dol`` package provide modules that perform the legacy DoL localizations that are performed
by the GBM Science Team. These python modules perform the operations of the original Fortran code.
